# Examples

Examples are application-oriented workflows. They are less pedagogical than tutorials and are meant to show complete physical models, scripts, and variations that users can adapt.

!!! tip "Tutorials teach; examples instantiate"
    If you are learning the package, start with the [tutorials](../tutorials/index.md). If you already know the workflow and want a physical model to adapt, start here.
