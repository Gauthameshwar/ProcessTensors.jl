Examples

Examples are application-oriented workflows. They are less pedagogical than tutorials and are meant to show complete physical models, scripts, and variations that users can adapt.

Tutorials teach; examples instantiate

If you are learning the package, start with the tutorials. If you already know the workflow and want a physical model to adapt, start here.

ExampleUse it for
Unitary spin chainA compact spin-chain unitary evolution script.
Bose-Hubbard dynamicsBosonic lattice dynamics with a local Hilbert-space cutoff.
